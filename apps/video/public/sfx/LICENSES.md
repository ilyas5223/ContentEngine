# SFX license inventory

Every clip in `apps/video/public/sfx/` MUST be listed here with its source and license.

## Adding a clip

1. Download from **Freesound** CC0 only (https://freesound.org/browse/tags/cc0/).
2. Save with the SFX name: `whoosh.mp3`, `pop.mp3`, `ding.mp3`.
3. Add an entry to the table below.
4. Register in `apps/video/src/audio/manifest.ts` under `SFX`.
